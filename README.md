<p align="center">

  <br/>
</p>

### How to use the boilerplate

```
1. Install ignite-cli if you don't have it

    $ npm install -g ignite-cli
    $ ignite new MobxApp -b mobx-boilerplate
    $ cd MobxApp

2. Run Build for either OS

    # for iOS
    $ react-native run-ios

    # for Android
    $ react-native run-android
```
